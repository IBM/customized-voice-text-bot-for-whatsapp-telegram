Twilio Sandbox

Twilio Sandbox is a prototyping tool that Twilio has provided for testing WhatsApp integrated applications. Here is how it works.

There is a single Twilio phone number for the sandbox: +1-415-523-8886. All Sandbox users share this same number, and then there is a specific "join..." message for each individual sandbox. Through this sandbox, Twilio can send templated messages to WhatsApp users, or reply to incoming messages: a WhatsApp user first sends this "join..." message to the Twilio sandbox phone number on WhatsApp to join a specific sandbox, and after that the user can send more messages. Twilio then forwards the incoming messages to an application through a webhook.

Steps for creating a Enabled Whatsapp Twilio account

  • Access the Twilio website and create an account by providing the requested information, as the following image shows:


  • Register a phone number into your created account, to start the free trial.


  • Then, set the following information for your account and click on Get Started:

    • Which Twilio product are you here to use? WhatsApp
    What do you plan to build with Twilio? IVR & Bots
    How do you want to build with Twilio? With code
    What is your preferred coding language? Python
    Would you like Twilio to host your code? No

  • And for last, activate your sandbox to enable all resources from your account.

Done! Now you are able to use your Twilio Account for testing the bot.
